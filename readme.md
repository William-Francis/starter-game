I'd like to create a very simple online game using HTML, Typescript with Node.js and Socket.io.
It should be a very basic top down game where players can join and write in their name and choose a color.
From here they will be represented by a small circle on the screen which they can control with W,A,S,D.
There will be random food shown as smaller green circles which spawn and players can collect these.
For every food a player collects, there will be a tail which follows the player. If a player collides with another players tail, they will stop moving for 4 seconds and loose all their food.
